ImagesFromWord Plugin for CKEditor

Introduction

This is a plugin for CKEditor that enhances SimpleUploads to detect Images pasted from MS Word.

Author:

Uritec SL

Version history:

This version: 1.1.2

Check online for updates and full history

Installation

1. Copying the files

Extract the contents of the zip in you plugins directory, so it ends up like this

ckeditor\
	...
	images\
	lang\
	plugins\
		...
		imagesfromword\
			plugin.js
			dialogs\
			docs\
			lang\
		...
	skins\
	themes\

2. Adding it to CKEditor

Now add the plugin in your config.js or custom js configuration file: config.extraPlugins='imagesfromword'; If you're already using other extraPlugins, you must add it to the same setting, adding a comma: config.extraPlugins='simpleuploads,imagesfromword';

If you have problems, please check the FAQ

3. Configuration

Currently there are no configuration options for this plugin

4. Translations

To add a new translation open plugin.js and search for "translations", it's at line 14 and add your language (the example is based on adding Polish 'pl' ). // translations lang : 'en,es', to // translations lang : 'en,es,pl',

Then in the lang folder copy the en.js to pl.js, edit that file and in the first line set your language: CKEDITOR.addPluginLang( "imagesfromword", "pl", and finally, translate all the text between quotes :-)

If you send it back to us, we will include it in the next versions.
